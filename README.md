# Calculator

This is a simple calculator made with VueJS

#### Project setup

```
npm install
```

#### Development mode

```
npm run serve
```

Development live preview is available at: [http://localhost:8080](http://localhost:8080)

#### Build for production

```
npm run build
```

#### Lints and fixes files

```
npm run lint
```

#### Format files

```
npm run format
```
